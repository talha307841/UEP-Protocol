# Changelog for uep-core

## [Unreleased]

## [0.1.0] - 2023-10-01
### Added
- Initial release of the Universal Exchange Protocol (UEP) system.
- Implemented core components including server, client, and message dispatcher.
- Added support for multiple transport methods: HTTP, gRPC, and WebSocket.
- Created adapters for integrating AI models and microservices.
- Defined UEP protocol version 1 with message formats and service definitions.
- Established logging and configuration management.

## [0.1.1] - 2023-10-15
### Changed
- Improved error handling in the message dispatcher.
- Enhanced logging format for better readability.
- Updated documentation for API and protocol specifications.

## [0.1.2] - 2023-10-30
### Fixed
- Resolved issues with message routing in the dispatcher.
- Fixed bugs in the gRPC transport implementation.
- Corrected minor typos in documentation files.

## [0.1.3] - 2023-11-15
### Added
- Introduced unit tests for core components and transport methods.
- Added integration tests for end-to-end functionality.
- Enhanced examples for better clarity and usage instructions.

## [0.1.4] - 2023-11-30
### Changed
- Refactored code for improved modularity and maintainability.
- Updated dependencies in requirements.txt for security patches.

## [0.1.5] - 2023-12-15
### Fixed
- Addressed performance issues in the message dispatcher.
- Fixed compatibility issues with certain AI model integrations.

## [0.1.6] - 2024-01-01
### Added
- Implemented support for additional transport protocols.
- Expanded documentation to include advanced usage scenarios and best practices.
